Luv user guide
==============

This guide is a small set of tutorials on using Luv_, a binding from OCaml to
libuv_, which is a high-performance asynchronous I/O library, and which offers
the same API on Windows and Unix.

.. rubric:: API reference

See the `generated API docs`_ for details on every module, function, and
type, with links to libuv documentation and man pages.

.. _Luv: https://github.com/aantron/luv
.. _libuv: https://github.com/libuv/libuv
.. _generated API docs: https://aantron.github.io/luv/luv/index.html
